# Sponsor Logos

Place sponsor logo images in this folder:

- `eti-logo.png` - ETI logo
- `trillium-logo.png` - Trillium Healthcare logo  
- `nc-dept-it-logo.png` - NC Department of IT logo

## Logo Requirements

- Recommended format: PNG with transparent background
- Recommended size: 200-300px width (height will scale proportionally)
- File names must match exactly: `eti-logo.png`, `trillium-logo.png`, `nc-dept-it-logo.png`

If logos are not available, the sponsor names will display as text fallback.
